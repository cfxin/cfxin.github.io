主备工作

1.1 安装Git

Git是版本控制工具，使用Git把本地的文件上传到GitHub。在Git官网下载需要的版本，安装选项全部默认即可。在最后一步添加路径时选择Use Git from the Windows Command Prompt，这样可以直接在命令提示符里打开git了。安装完成后在命令提示符中输入git –version验证是否安装成功。
